REQUIREMENTS:
-------------------------------------------------
- Java JDK 8 atau lebih baru
- Sistem Operasi: Windows / Mac / Linux

-------------------------------------------------
CARA MENJALANKAN PROGRAM:
-------------------------------------------------
1. Pastikan Java sudah terinstall di komputer
   Cek dengan perintah: java -version

2. Buka terminal / command prompt

3. Masuk ke folder tempat file berada:
   cd [nama-folder]

4. Compile program:
   javac SistemPasienGUI.java

5. Jalankan program:
   java SistemPasienGUI

6. Tampilan GUI akan muncul secara otomatis

-------------------------------------------------
FITUR PROGRAM:
-------------------------------------------------
- Tambah data pasien baru
- Edit data pasien
- Hapus data pasien (soft delete)
- Cari pasien berdasarkan Nama
- Cari pasien berdasarkan ID
- Sort data berdasarkan Nama (Selection Sort)
- Sort data berdasarkan ID (Bubble Sort)
- Sort data berdasarkan Total Biaya
- Statistik data pasien
